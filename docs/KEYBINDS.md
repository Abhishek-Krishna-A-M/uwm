# UWM Keybindings

All bindings use **Super (Logo/Mod4)** as the primary modifier.

## Launchers

| Binding | Action |
|---------|--------|
| `Super+Return` | Launch foot |
| `Super+r` | Launch fuzzel |

## Navigation

| Binding | Action |
|---------|--------|
| `Super+h` | Focus left |
| `Super+j` | Focus down |
| `Super+k` | Focus up |
| `Super+l` | Focus right |
| `Super+Tab` | Switch to previous workspace |
| `Super+c` | Focus previous window (last focused) |

## Workspace Management

| Binding | Action |
|---------|--------|
| `Super+1`..`Super+9` | Switch to workspace N |
| `Super+Shift+1`..`Super+Shift+9` | Move focused window to workspace N |

## BSP Layout

| Binding | Action |
|---------|--------|
| `Super+Shift+h` | Swap with left window |
| `Super+Shift+j` | Swap with window below |
| `Super+Shift+k` | Swap with window above |
| `Super+Shift+l` | Swap with right window |
| `Super+Shift+r` | Rotate split direction |
| `Super+Alt+h` | Resize (shrink toward left) |
| `Super+Alt+j` | Resize (shrink downward) |
| `Super+Alt+k` | Resize (grow upward) |
| `Super+Alt+l` | Resize (grow rightward) |

## Container Layouts

| Binding | Action |
|---------|--------|
| `Super+m` | Toggle tabbed container |
| `Super+Shift+m` | Toggle monocle container |
| `Super+t` | Set BSP mode |

## Window Operations

| Binding | Action |
|---------|--------|
| `Super+Shift+f` | Toggle floating |
| `Super+f` | Toggle fullscreen |
| `Super+w` | Close focused window |
| `Super+Shift+w` | Force-close focused window |

## System

| Binding | Action |
|---------|--------|
| `Super+Escape` | Quit UWM |
